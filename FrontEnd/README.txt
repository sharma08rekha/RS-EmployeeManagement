Please flow below steps to set up the Employment management UI project.

1) As a prerequisite Install NodeJs.
2) Open the Node command line and go to the project base directory.
4) Use command npm install to install the required packages.
5) Use the command npm start to start the Angular application.
6) NPM start will open the app uding URL http://localhost:4200
7) At home page will load the Employee list, details components as part of App componet.
8) On load on Employee list, Get call will be made to backend service and UI will show all the existing employees if any.
8) User can Add,Update and Delete the employees.
